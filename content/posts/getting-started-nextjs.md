## Why Next.js?

Next.js is a React framework that gives you **server-side rendering**, **static site generation**, and a fantastic developer experience out of the box.

### Key Features

- **App Router** — file-system based routing with layouts
- **Server Components** — render on the server by default
- **Streaming** — progressive rendering with Suspense
- **Built-in optimizations** — fonts, images, scripts

### Quick Start

```bash
pnpm create next-app@latest my-app --typescript --tailwind --app
cd my-app
pnpm dev
```

Visit `localhost:3000` and start building!
